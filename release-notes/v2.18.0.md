## mqvi v2.18.0

### New Features

- **push**: mobile push notifications for DMs and incoming calls on Android and iOS
- **push**: native incoming-call wake — Android full-screen intent and iOS CallKit/PushKit, with answerable call pushes
- **push**: iOS Push Notifications capability, VoIP background mode, and direct APNs delivery for DM alerts
- **admin**: sortable feedback ticket datagrid with per-admin unread indicators and multi-select status/type filters
- **admin**: multi-select filters on the Servers and Users panels (state, presence, admin, managed/self)
- **roles**: reworked role settings — owner role is mentionable, a separate default-permissions entry, and clearer administrator-override behavior
- **chat**: reflow messages by column width instead of window width, keeping split-view and narrow layouts readable
- **chat**: bottom-sheet emoji/gif pickers on narrow columns and an emoji picker in edit mode
- **chat**: drag-to-resize handle on the message input
- **settings**: slide-in navigation drawer on mobile, replacing the horizontal tab bar
- **connections**: server switcher is now a modal reachable on mobile and before login
- **voice**: enforce voice permission changes mid-call
- **ws**: per-connection inbound event rate limiting
- **legal**: public delete-account page with a 30-day grace period

### Bug Fixes

- **security**: enforce server ownership on server-scoped endpoints and close cross-server IDOR, invite-waste, WS log-flood, and voice-lock findings
- **voice**: enforce server-mute at the SFU so it can't be bypassed
- **voice**: disconnect participants when a voice channel or server is deleted
- **auth**: keep tokens on transient failures and force-logout only on refresh rejection
- **ws**: serialize per-connection inbound events to prevent dropped voice state
- **ws**: prevent a send-on-closed-channel panic on client teardown
- **invite**: make invite consumption atomic and stop leaking internals on errors
- **push**: forward APNs registration to Capacitor for iOS DM tokens and stop the ringing device when a call is cancelled
- **settings**: make the two-column admin panels usable on mid-size screens
- **chat**: size the edit box to the message instead of a fixed height
- **invite**: load the logo from an absolute path on the web
- **legal**: restore broken legal pages and add moderation and push-token disclosures
- **android**: drop unused camera and media-playback foreground service permissions
